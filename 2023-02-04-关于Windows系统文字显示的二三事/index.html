# еӯҳеңЁй—®йўҳ

Windows зі»з»ҹиҮӘеёҰзҡ„еҫ®иҪҜйӣ…й»‘еӯ—дҪ“иҮӘ 1997е№ҙ7жңҲ18ж—Ҙ еҸ‘иЎҢиө·е·Із»ҸжІҝз”ЁдәҢеҚҒдә”е№ҙпјҢжңҹй—ҙжҳҫзӨәи®ҫеӨҮеҸ‘еұ•иҝ…йҖҹгҖӮжҳҫз„¶пјҢиҝҷж¬ҫеӯ—дҪ“еҚідҪҝз»ҸиҝҮдјҳеҢ–жӣҙж–°пјҢдҪҶиҝҳжҳҜйҖӮеә”дёҚдәҶеҪ“д»Ҡзҡ„жҳҫзӨәи®ҫеӨҮпјҢе°Өе…¶жҳҜ 4k жҳҫзӨәеҷЁпјҢеҠ д№Ӣ Windows зі»з»ҹеҜ№еӯ—дҪ“зіҹзі•зҡ„жёІжҹ“дә§з”ҹзҡ„й”ҜйҪҝпјҢеҸҜжүҖи°“жғЁдёҚеҝҚзқ№.

# дҝ®ж”№зі»з»ҹеӯ—дҪ“

еңЁзҪ‘дёҠжҹҘиҜўеҲ°зҡ„еӨ§еӨҡж•°дҝ®ж”№зі»з»ҹеӯ—дҪ“зҡ„ж–№жі•йғҪжҳҜдҝ®ж”№жіЁеҶҢиЎЁпјҢиҝҷз§Қж–№ејҸдёҚд»…йә»зғҰпјҢеҜ№е°ҸзҷҪдёҚеҸӢеҘҪпјҢиҖҢдё”е°ұз®—еҜ№иҖҒйёҹжқҘиҜҙпјҢд№ҹдёҚжғіиҪ»жҳ“дҝ®ж”№жіЁеҶҢиЎЁгҖӮеҘҪеңЁйқўеҜ№еӯ—дҪ“й—®йўҳзҡ„дёҚеҸӘжҳҜе’ұд»¬пјҢж—Ҙжң¬жңүдҪҚиҖҒе“Ҙе°ұејҖеҸ‘дәҶдёӘжҜ”иҫғеҘҪзҡ„иҪҜд»¶пјҢеҸҜд»Ҙж–№дҫҝзҡ„жӣҙж”№ж“ҚдҪңзі»з»ҹеӯ—дҪ“.

# жҢ‘йҖүзі»з»ҹеӯ—дҪ“

иҝҷдёӘжң¬жқҘжІЎе•ҘеҘҪиҜҙзҡ„пјҢзңӢиҮӘе·ұе–ңеҘҪпјҢдёҚиҝҮеҘҪзҡ„еӯ—дҪ“зЎ®е®һжҢәйҡҫжүҫзҡ„гҖӮе’ұд№ӢеүҚдҪҝз”Ёзҡ„еӯ—дҪ“дёә Monaco пјҢиӢ№жһңеҮәе“Ғзҡ„дёҖж¬ҫеӯ—дҪ“пјҢеӯ—дҪ“жө‘еңҶйҘұж»ЎпјҢеҸҜд»Ҙж»Ўи¶іеҲҶжё… 0 е’Ң O зӯүзј–зЁӢеӯ—дҪ“еҹәжң¬иҰҒжұӮгҖӮиҝҷжҳҜеңЁзңӢзҝҒеҮҜиҖҒеёҲзҡ„ c иҜӯиЁҖиҜҫж—¶еҸ‘зҺ°зҡ„пјҢе’ұдёҖдёӢе°ұе–ңж¬ўдёҠдәҶгҖӮдҪҶжҳҜпјҢиҝҷжҳҜдёҖж¬ҫиӢұж–Үеӯ—дҪ“пјҢдёҚеҢ…еҗ«дёӯж–ҮгҖӮеӣ жӯӨжҳҫзӨәдёӯж–ҮйӮЈе°ұжҳҜдҫқжүҳзӯ”иҫ©пјҢд№ӢеүҚйғҪж•ҙз®—жі•дёҚжҖҺд№Ҳж•Ідёӯж–Үд№ҹе°ұеҮ‘еҗҲпјҢдҪҶдёӯж–ҮеӨҡдәҶе°ұеҝҚдёҚдәҶгҖӮеӣ жӯӨе’ұе°ұйӯ”ж”№дәҶдёҖдёӢрҹҳӢгҖӮ
