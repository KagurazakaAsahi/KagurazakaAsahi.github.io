йҰ–йЎө еӯҰд№  JAVAе…«иӮЎж–Ү

# java зәҝзЁӢжұ жңүе“Әдәӣж ёеҝғеҸӮж•°пјҢеҲҶеҲ«жңүд»Җд№ҲдҪңз”Ё

жһ„йҖ ж–№жі•жңҖеӨҡзҡ„жҳҜдёғдёӘеҸӮж•°

# зәҝзЁӢжұ жңүе“ӘдәӣжӢ’з»қзӯ–з•Ҙ

JDK жҸҗдҫӣдәҶеӣӣз§ҚеҶ…зҪ®зҡ„жӢ’з»қзӯ–з•Ҙ

  • AbortPolicy (й»ҳи®Ө): дёўејғд»»еҠЎе№¶жҠӣеҮә RejectedExecutionException ејӮеёё
  • DiscardPolicy: зӣҙжҺҘдёўејғд»»еҠЎпјҢдёҚжҠӣеҮәејӮеёёпјҢжІЎжңүд»»дҪ•жҸҗзӨә
  • DiscardOldestPolicy: дёўејғд»»еҠЎйҳҹеҲ—дёӯжңҖйқ еүҚзҡ„д»»еҠЎпјҢеҪ“еүҚжҸҗдәӨзҡ„д»»еҠЎдёҚдјҡдёўејғ
  • CallerRunsPolicy: дәӨз”ұд»»еҠЎзҡ„и°ғз”ЁзәҝзЁӢ (жҸҗдәӨд»»еҠЎзҡ„зәҝзЁӢ) жқҘжү§иЎҢд»»еҠЎ

йҷӨд»ҘдёҠеӣӣз§ҚжӢ’з»қзӯ–з•ҘпјҢиҝҳеҸҜд»ҘйҖҡиҝҮе®һзҺ° RejectedExecutionHandler жҺҘеҸЈпјҢе®һзҺ°иҮӘе®ҡд№үзҡ„жӢ’з»қзӯ–з•Ҙ

# иҜҙдёҖиҜҙзәҝзЁӢжұ зҡ„жү§иЎҢжөҒзЁӢ

# зәҝзЁӢжұ ж ёеҝғзәҝзЁӢж•°иҜҘеҰӮдҪ•и®ҫзҪ®

Nthread=NcpuГ—UcpuГ—(1+W/C)N_{thread} =N_{cpu}\times U_{cpu}\times (1+W/C)NthreadвҖӢ=NcpuвҖӢГ—UcpuвҖӢГ—(1+W/C)

  • Ncpu = cpu зҡ„ж ёеҝғж•°пјҢUcpu = cpu зҡ„дҪҝз”ЁзҺҮ (еңЁ 0~1 д№Ӣй—ҙ)
  • W = зәҝзЁӢзӯүеҫ…ж—¶й—ҙпјҢC = зәҝзЁӢи®Ўз®—ж—¶й—ҙ

ж №жҚ®д»ҘдёҠе…¬ејҸпјҢеҸҜе°Ҷд»»еҠЎеҲҶдёә CPU еҜҶйӣҶеһӢе’Ң IO еҜҶйӣҶеһӢ

# CPU еҜҶйӣҶеһӢ

CPU еҜҶйӣҶеһӢпјҡзәҝзЁӢж•° = CPU ж ёеҝғж•° + 1

иҝҷз§Қд»»еҠЎдё»иҰҒж¶ҲиҖ— CPU иө„жәҗпјҢжҜ”еҰӮеҠ и§ЈеҜҶпјҢеҺӢзј©пјҢи®Ўз®—зӯүдёҖзі»еҲ—йңҖиҰҒеӨ§йҮҸиҖ—иҙ№ CPU иө„жәҗзҡ„д»»еҠЎ

жҜ” CPU ж ёеҝғж•°еӨҡеҮәжқҘзҡ„дёҖдёӘзәҝзЁӢжҳҜдёәдәҶйҳІжӯўзәҝзЁӢеҒ¶еҸ‘зҡ„зјәйЎөдёӯж–ӯпјҢжҲ–иҖ…е…¶д»–еҺҹеӣ еҜјиҮҙзҡ„д»»еҠЎжҡӮеҒңиҖҢеёҰжқҘзҡ„еҪұе“ҚгҖӮдёҖж—Ұд»»еҠЎжҡӮеҒңпјҢCPU е°ұдјҡеӨ„дәҺз©әй—ІзҠ¶жҖҒпјҢиҖҢиҝҷз§Қжғ…еҶөдёӢеӨҡеҮәжқҘзҡ„дёҖдёӘзәҝзЁӢе°ұеҸҜд»Ҙе……еҲҶеҲ©з”Ё CPU зҡ„з©әй—Іж—¶й—ҙ.

# IO еҜҶйӣҶеһӢ

зәҝзЁӢж•° = CPU ж ёеҝғж•° * 2

иҝҷз§Қд»»еҠЎдјҡжңүеӨ§йғЁеҲҶж—¶й—ҙиҝӣиЎҢ IO ж“ҚдҪңпјҢжҜ”еҰӮжғі MySQL ж•°жҚ®еә“пјҢж–Үд»¶иҜ»еҶҷпјҢзҪ‘з»ңйҖҡдҝЎзӯүд»»еҠЎпјҢиҝҷзұ»д»»еҠЎдёҚдјҡзү№еҲ«ж¶ҲиҖ— CPU иө„жәҗпјҢдҪҶжҳҜ IO ж“ҚдҪңжҜ”иҫғиҖ—ж—¶пјҢдјҡеҚ з”ЁжҜ”иҫғеӨҡзҡ„ж—¶й—ҙ

зәҝзЁӢеңЁеӨ„зҗҶ IO зҡ„ж—¶й—ҙж®өеҶ…дёҚдјҡеҚ з”Ё CPU, иҝҷж—¶е°ұеҸҜд»Ҙе°Ҷ CPU дәӨеҮәз»ҷе…¶д»–зәҝзЁӢдҪҝз”ЁпјҢеӣ жӯӨеңЁ IO еҜҶйӣҶеһӢд»»еҠЎзҡ„еә”з”ЁдёӯеҸҜд»ҘеӨҡй…ҚзҪ®дёҖдәӣзәҝзЁӢ

еӣ жӯӨж ёеҝғзәҝзЁӢж•°й…ҚзҪ®зҡ„ж•°йҮҸеҸӮиҖғ CPU зҡ„иҝҗиЎҢпјҢеҗҲзҗҶй…ҚзҪ®зәҝзЁӢж•°йҮҸе°ұеҸҜд»ҘжңҖеӨ§йҷҗеәҰзҡ„еҲ©з”Ё CPU

еҹәжң¬еҺҹеҲҷ:

  • зәҝзЁӢжү§иЎҢж—¶й—ҙи¶ҠеӨҡпјҢе°ұйңҖиҰҒи¶Ҡе°‘зҡ„зәҝзЁӢ
  • зәҝзЁӢжү§иЎҢж—¶й—ҙи¶Ҡе°‘пјҢе°ұйңҖиҰҒи¶ҠеӨҡзҡ„зәҝзЁӢ
жӣҙж–°дәҺ 2024-09-05 йҳ…иҜ»ж¬Ўж•° ж¬Ў
